Web-интерфейс для проекта SQL3. Приложение с использованием MVC-фреймворка (Spring). Приложение поддерживает CRUD-операций, импорт/экспорт таблиц, формирование "персональных предложений" через графический интерфейс, поддерживает авторизацию пользователей по ролям (Администратор, Посетитель), а также логирование действий пользователей.
